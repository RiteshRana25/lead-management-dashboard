## Setup Instructions

### 1. Clone the repository

git clone <your-repo-url>
cd <your-repo-folder>

### 2. frontend setup

cd lead-management-dashboard

npm install
npm run dev

### 3. backend setup

cd lead-management-dashboard
cd backend
